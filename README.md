# 📱 Beautiful Auth UI

A modern and elegant authentication UI built with **React Native**, **Expo**, and **Expo Router**. This project showcases a clean authentication flow with responsive layouts, smooth styling, and reusable components.

---

## ✨ Features

- 🔐 Login Screen
- 📝 Sign Up Screen
- 🚀 Splash Screen
- 📱 Fully Responsive UI
- 🎨 Modern & Minimal Design
- ⚡ Expo Router Navigation
- 🌈 Linear Gradient Backgrounds
- 🧩 Reusable Input Components
- 📐 Responsive Utility Functions (RW, RH, RF, RS)

---

## 🛠️ Tech Stack

- Expo
- Expo Router
- TypeScript
- Expo Linear Gradient
- Expo Vector Icons
